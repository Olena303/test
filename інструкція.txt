Для запуску проекту Вам потрібно мати встановлений composer та NoteJS
А також налаштувати Ваш сервер на виконання файлу index.php
1) В дерикторії проекту потрібно виконааті команду  npm install
2) В дерикторії проекту потрібно виконааті команду  composer install
3) В дерикторії проекту потрібно виконааті команду npx sass style.scss style.css
4) Відкрийте файл handler.php та в зміних для налаштування SMTP сервера -  введіть свої дані. (Можете цього не робити так як я внесла свої налаштування для тестування)

Після чого переходьте на локальну адресу яку ви вказали на сервері та користуйтесь скриптом.
